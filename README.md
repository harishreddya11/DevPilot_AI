# DevPilot AI

DevPilot AI is an AI-powered developer workspace designed to help developers write, review, debug, and understand code more efficiently.

Unlike a general-purpose chatbot, DevPilot AI focuses on software development workflows by combining AI with developer tools in a modern web application.

## Features

- AI Chat
- Code Generation
- Code Explanation
- Code Review
- Bug Detection
- Documentation Generation
- SQL Assistant
- Resume Review
- Interview Preparation
- Project Analysis

## Tech Stack

### Frontend
- React
- Vite
- Tailwind CSS

### Backend
- FastAPI
- PostgreSQL
- SQLAlchemy
- OpenRouter API
